Understanding Color Models
  • Color by Name: You can use predefined color names such as "red", "blue", etc.
  • Hexadecimal (Hex): A six-character string starting with "#" (e.g., #ff6347), where the first two characters represent red, the next two represent green, and the last two represent blue.
  • RGB: Uses a combination of Red, Green, and Blue, with values ranging from 0 to 255 (e.g., rgb(255, 0, 0) for red).
  • RGBA: Like RGB but includes an alpha channel for transparency (e.g., rgba(255, 0, 0, 0.5) is a semi-transparent red).
  • HSL: Defines colors by Hue (0°-360°), Saturation (0%-100%), and Lightness (0%-100%) (e.g., hsl(120, 100%, 50%) for green).
  • HSLA: Like HSL but includes an alpha channel for transparency (e.g., hsla(120, 100%, 50%, 0.5) is a semi-transparent green).
  • Hex with Alpha: A hexadecimal code with an 8-character format that includes alpha transparency (e.g., #ff634780). The last two characters represent alpha (e.g., 80 = 50% opacity).
