A multi-section, responsive blog/agency landing page built using **Bootstrap 5** and custom CSS/SASS for styling. This project demonstrates proficiency in utilizing the Bootstrap framework for rapid and responsive web development.

## âœ¨ Features

- **Fully Responsive Design:** Optimized for desktop, tablet, and mobile viewing using Bootstrap's grid system.
- **Sticky Navigation Bar:** The main navigation remains fixed at the top, improving user experience.
- **Multi-Section Layout:** Includes dedicated sections for:
  - Hero (Header)
  - Services
  - Portfolio (Gallery)
  - About Us
  - Team
  - Contact Form
- **Bootstrap Components:** Effective use of Navbar, Grid System, Buttons, and Utility classes.
- **Bootstrap Icons:** Integrated for a modern and visually appealing interface.
- **Custom Styling:** Enhanced visual design with custom CSS variables and custom styles defined in `styles.css`.
